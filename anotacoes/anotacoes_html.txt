target =
_blank - Abre o documento vinculado numa nova janela ou guia
_self - Abre o documento vinculado no mesmo quadro em que foi clicado (isso é padrão)
_parent - Abre o documento vinculado no quadro dos pais
_top - Abre o documento vinculado em todo o corpo da janela
framename - Abre o documento vinculado no iframe nomeado

--------------------------------------------------------------

rel = 

alternate - Fornece um link para uma representação alternativa do documento (ou seja, página de impressão, traduzida ou espelho)

author - Fornece um link para o autor do documento

bookmarking - URL permanente usado para bookmarking

external - Indica que o documento referenciado não faz parte do mesmo site que o documento atual

help - Fornece um link para um documento de ajuda

license - Fornece um link para informações de licenciamento para o documento

next - Fornece um link para o próximo documento da série

nofollow - Links para um documento não endossado, como um link pago.("nofollow" é usado pelo Google, para especificar que o spider de pesquisa do Google não deve seguir esse link)

noopener - Exige que qualquer contexto de navegação criado seguindo o hiperlink não deve ter um contexto de navegação de abertura

noreferrer - Torna o referenciador desconhecido. Nenhum cabeçalho de árbitro será incluído quando o usuário clicar no hiperlink

prev - O documento anterior em uma seleção

search - Links para uma ferramenta de pesquisa para o documento

tag - Uma tag (chave) para o documento atual
